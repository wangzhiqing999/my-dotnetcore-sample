命令行下面，执行 dotnet new mvc 命令创建一个 web 应用

执行：

    dotnet new mvc

运行完毕后， 在当前目录，生成一系列的目录与文件.


	
运行 Visual Studio Code，选择【文件】下面的【打开文件夹】。打开当前目录。

选择  Startup.cs 文件。

顶部出现一个【警告】提示框与一个【信息】提示框。

【警告】提示框， 选择 【Yes】
【信息】提示框，选择【Restore】


按 F5 运行。


具体操作参考：

    https://docs.microsoft.com/zh-cn/aspnet/core/tutorials/first-mvc-app-xplat/start-mvc
